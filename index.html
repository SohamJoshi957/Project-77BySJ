First, you must download visual studio. Secondly, go to "File Explorer" and then "Desktop" inside File Explorer. Next, create a new folder, and rename it to anything you want. Next, Open Visual Studio, and click on "File" on the top left corner. Once you click on it, click on "Open New Folder", and open the folder you created in your desktop. On the left hand side, you can see the explorer. Right click on the blank grey space and click "New File". The 1st time, rename it to "index.html". Next time, make a new file and rename it it "style.css". Lastly, make another file and rename it to "main.js" You are ready to start coding!!!

Console application (app) is a program developed in Visual Studio which accepts input parameter, calls the required service, runs business logic and sends output to the console, this console is known as the command prompt
